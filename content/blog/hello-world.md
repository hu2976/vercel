# 我的第一篇博客

欢迎来到我的新博客！🎉

## 为什么要搭建这个网站？

我一直想要一个属于自己的空间来记录学习和思考。这个网站使用以下技术栈构建：

- **React 18** — 用户界面框架
- **Vite** — 极速构建工具
- **Tailwind CSS** — 实用优先的 CSS 框架
- **Markdown** — 博客文章格式
